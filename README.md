# 📊 Smartphone Impact Analysis

This project analyzes the relationship between smartphone usage, sleep patterns, stress levels, and productivity using a real-world dataset. The goal is to uncover insights and understand how screen time affects daily life.

---

## 🎯 Objectives

- Analyze smartphone usage patterns
- Explore relationships between:
  - 📱 Screen time
  - 😴 Sleep hours
  - 😰 Stress levels
  - 📈 Productivity
- Perform Exploratory Data Analysis (EDA)
- Visualize key trends and correlations

---

## 🧠 Technologies Used

- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn

---

## 📊 Dataset

This project uses a dataset containing information such as:

- Age
- Gender
- Occupation
- Daily phone usage
- Social media usage
- Sleep hours
- Stress level
- Productivity score

---

## 🔍 Project Workflow

1. Data loading and preprocessing  
2. Handling missing values  
3. Exploratory Data Analysis (EDA)  
4. Visualization of key variables  
5. Correlation analysis  
6. Insights and conclusions  

---

## 📈 Key Insights

- Increased screen time is associated with reduced sleep duration  
- Higher social media usage correlates with increased stress  
- Productivity tends to decrease with excessive phone usage
